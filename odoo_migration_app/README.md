# Odoo Migration App

This app helps you migrate data between different versions of Odoo. It provides functionalities like:

- Connecting to multiple Odoo instances
- Selecting models and fields to migrate
- Mapping fields between versions
- Testing migrations for single records
- Handling errors and retries
- Logging and document management

## Installation

1. Clone the repository.
2. Place the app folder in your Odoo `addons` directory.
3. Install the app from the Odoo interface.

## Usage

- Connect to the source and destination Odoo instances.
- Select the models and fields to migrate.
- Test with specific records and perform migrations.
- Handle errors and retries through the app interface.
